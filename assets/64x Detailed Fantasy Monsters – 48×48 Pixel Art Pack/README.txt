FILE FORMAT
-----------

All monster sprites are provided as transparent PNG files.

Each sprite was created on a 48x48 pixel canvas.

For the sharpest results, use nearest-neighbor interpolation when scaling the sprites.

Recommended scaling examples:

- 48x48 pixels
- 96x96 pixels
- 144x144 pixels
- 192x192 pixels
- 240x240 pixels

Avoid smooth or bilinear filtering because it may blur the pixel art.

INSTALLATION
------------

1. Extract the ZIP file.
2. Open the folder containing the monster PNG files.
3. Import the sprites into your game engine or graphics software.
4. Disable texture filtering if necessary.
5. Use nearest-neighbor scaling to preserve the pixel art.


ENGINE IMPORT TIPS
------------------

GODOT

- Import the PNG files into your project.
- Select the sprites in the FileSystem panel.
- Disable Filter in the import settings.
- Reimport the textures.
- Use integer scaling whenever possible.

UNITY

- Set Texture Type to Sprite (2D and UI).
- Set Filter Mode to Point.
- Set Compression to None.
- Set Pixels Per Unit according to your project.
- Apply the import settings.

GAMEMAKER

- Import the PNG files as sprites.
- Disable interpolation between pixels.
- Keep the sprite scale at whole-number values.

RPG MAKER

- Import the files into the appropriate image folder.
- Resize or arrange the sprites according to your battle system and project requirements.
